.. function:: make_confusion_matrix(cf, group_names=None, categories='auto', count=True, percent=True, cbar=True, xyticks=True, xyplotlabels=True, sum_stats=True, figsize=None, cmap='Blues', title=None)

   This function will make a pretty plot of an sklearn Confusion Matrix cm using a Seaborn heatmap visualization.

   Method borrowed from https://github.com/DTrimarchi10/confusion_matrix/blob/master/cf_matrix.py

   :param cf:
   :type cf: confusion matrix to be passed in
   :param group_names:
   :type group_names: List of strings that represent the labels row by row to be shown in each square.
   :param categories:
   :type categories: List of strings containing the categories to be displayed on the x,y axis. Default is 'auto'
   :param count:
   :type count: If True, show the raw number in the confusion matrix. Default is True.
   :param percent:
   :type percent: If True, show the proportions for each category. Default is True.
   :param cbar: Default is True.
   :type cbar: If True, show the color bar. The cbar values are based off the values in the confusion matrix.
   :param xyticks:
   :type xyticks: If True, show x and y ticks. Default is True.
   :param xyplotlabels:
   :type xyplotlabels: If True, show 'True Label' and 'Predicted Label' on the figure. Default is True.
   :param sum_stats:
   :type sum_stats: If True, display summary statistics below the figure. Default is True.
   :param figsize:
   :type figsize: Tuple representing the figure size. Default will be the matplotlib rcParams value.
   :param cmap: See http://matplotlib.org/examples/color/colormaps_reference.html
   :type cmap: Colormap of the values displayed from matplotlib.pyplot.cm. Default is 'Blues'
   :param title:
   :type title: Title for the heatmap. Default is None.

   :returns:
   :rtype: handle to current figure

   Create plot for the Receiver Operating Characteristic (ROC) curve

   :param y_label: ground truth labels

   :param y_score: 1D array with the confidence scores from the model predictions
   :type y_score: np.array
   :param selected_thresh: what threshold you want to highlight
   :type selected_thresh: float
   :param ax: optional, plots on given axis, or creates new figure otherwise
   :type ax: AxesSubplot

   :returns:
